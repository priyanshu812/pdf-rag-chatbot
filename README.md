📄 Chat with PDF — RAG Based AI Document Assistant

An AI-powered web application that allows users to upload a PDF and ask questions about its content.
The system uses Retrieval-Augmented Generation (RAG) to retrieve relevant information from the document and generate accurate responses using Google Gemini.

Instead of sending the entire document to the LLM, the system retrieves only the most relevant sections using vector similarity search with FAISS, making responses faster, cheaper, and more reliable.

⸻

🚀 Features
	•	📄 Upload and process PDF documents
	•	💬 Ask natural language questions about the document
	•	🔍 Semantic search using vector embeddings
	•	🧠 Context-aware answers generated by Gemini LLM
	•	💾 Efficient document indexing using FAISS
	•	🗂 Conversation history for follow-up questions
	•	⚡ Interactive chat interface built with Streamlit

⸻

🧠 How the System Works (RAG Pipeline)

The application follows a Retrieval-Augmented Generation architecture.

1️⃣ Document Processing (Indexing Phase)

When a PDF is uploaded:

PDF File
   ↓
Text Extraction (PyPDF2)
   ↓
Text Chunking (RecursiveCharacterTextSplitter)
   ↓
Embedding Generation (Gemini Embedding Model)
   ↓
Vector Storage (FAISS)

The document is converted into vector embeddings and stored in a FAISS vector database.

⸻

2️⃣ Question Answering (Retrieval Phase)

When the user asks a question:

User Question
   ↓
Question Embedding
   ↓
FAISS Similarity Search
   ↓
Retrieve Top Relevant Chunks
   ↓
Build Prompt (Context + Chat History)
   ↓
Gemini LLM Generates Answer

Only the most relevant document chunks are sent to the LLM, reducing hallucinations and improving response accuracy.

⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/priyanshu812/pdf-rag-chatbot.git
cd pdf-rag-chatbot


⸻

2️⃣ Create a virtual environment

python -m venv venv
source venv/bin/activate


⸻

3️⃣ Install dependencies

pip install -r requirements.txt


⸻

4️⃣ Add your Gemini API Key

Create a .env file:

GOOGLE_API_KEY=your_api_key_here

You can obtain an API key from:

https://ai.google.dev/

⸻

5️⃣ Run the application

streamlit run app.py

The app will start at:

http://localhost:8501
